# 📝 Lista de Tarefas em JavaScript

Minha primeira aplicação web usando JavaScript puro.

## 🚀 Funcionalidades

- Adicionar tarefas
- Remover tarefas
- Marcar tarefas como concluídas
- Limpar todas as tarefas
- Contador de tarefas
- Salvar tarefas no navegador (LocalStorage)
- Adicionar tarefas pressionando Enter

## 🛠 Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## 🎯 Objetivo do projeto

Este projeto foi criado como parte do meu aprendizado em desenvolvimento Full Stack.
