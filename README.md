# 🧠 Lista de Tarefas Kanban

Aplicação de gerenciamento de tarefas estilo Kanban desenvolvida com JavaScript puro.

## 🚀 Demonstração

https://pontesfil.github.io/lista-tarefas-js/

## 🛠 Tecnologias

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage

## ✨ Funcionalidades

- Criar tarefas
- Editar tarefas
- Remover tarefas
- Drag and Drop entre colunas
- Filtro de tarefas
- Persistência no navegador

## 🎯 Objetivo

Projeto desenvolvido para estudo de manipulação de DOM e lógica de aplicações frontend sem frameworks.

## 📚 Aprendizados

- Manipulação de eventos
- Drag and Drop API
- Persistência com LocalStorage
- Estrutura de aplicações JavaScript
